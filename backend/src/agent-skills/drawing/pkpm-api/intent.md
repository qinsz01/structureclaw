# PKPM Drawing Export

- `zh`: 当用户要求使用 PKPM 生成施工图时使用。支持平面图、立面图、剖面图、节点详图和配筋图等图纸类型，输出格式包括 DWG、DXF 和 PDF。需要上游结构模型和分析结果作为输入。
- `en`: Use when the request explicitly asks to generate construction drawings via PKPM. Supports plan, elevation, section, detail, and reinforcement drawing types. Output formats include DWG, DXF, and PDF. Requires upstream structural model and analysis results as input.
