# PNG 导出

本技能将当前三维结构场景截图并导出为 PNG 文件，支持以下功能：

## 导出模式

| 模式 | 触发词示例 | 说明 |
|------|-----------|------|
| 单工况截图 | "截图"、"保存图片" | 导出当前视口的当前工况 |
| 多工况批量导出 | "批量导出"、"all cases" | 循环遍历所有工况，每个工况导出一张 PNG |
| 自定义分辨率 | "高清导出"、"2x"、"4x" | 以 2 倍或 4 倍像素密度导出，适合报告插图 |

## 文件命名规则

```
{前缀}_{结构标题}_{工况名称}_{时间戳}.png
```

示例：`export_三层钢框架_恒荷载+活荷载_20260406-143022.png`

- **前缀**：可由用户在对话中指定（如"用 report 作前缀"），默认为 `export`
- **结构标题**：来自当前项目元数据
- **工况名称**：来自当前激活的荷载工况
- **时间戳**：`YYYYMMDD-HHmmss` 格式，防止文件名冲突

## 分辨率选项

| 选项 | 像素倍数 | 适用场景 |
|------|---------|---------| 
| 标准（1x） | 屏幕原始分辨率 | 快速预览 |
| 高清（2x） | 屏幕分辨率 × 2 | 报告插图、演示文稿 |
| 超高清（4x） | 屏幕分辨率 × 4 | 印刷级输出 |

> 批量导出时，所有图片将打包为单个 ZIP 文件下载。
